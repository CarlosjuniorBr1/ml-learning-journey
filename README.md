# ML Learning Journey

Repositório criado para documentar minha jornada de estudos em **Machine Learning**, reunindo anotações, implementações, experimentos e exercícios práticos.

## Objetivos

* Estudar os principais conceitos de Machine Learning.
* Reproduzir e compreender os exemplos do livro.
* Documentar aprendizados e boas práticas.
* Construir um portfólio público de estudos.

## Estrutura

```text
.
├── chapter-02/
├── chapter-03/
├── datasets/
├── README.md
└── requirements.txt
```

Cada capítulo contém os notebooks, anotações e experimentos desenvolvidos durante o estudo.

## Tecnologias

* Python
* NumPy
* Pandas
* Matplotlib
* Scikit-Learn
* TensorFlow
* Jupyter Notebook

## Referência

Os estudos são baseados principalmente no livro **Mãos à Obra: Aprendizado de Máquina com Scikit-Learn, Keras & TensorFlow (2ª edição)**, complementados por documentação oficial e experimentos próprios.

> Este repositório tem finalidade exclusivamente educacional. Todo o conteúdo, anotações e implementações foram produzidos durante meu processo de aprendizagem.
